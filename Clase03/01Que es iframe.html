Etiqueta iframe

Se utiliza para incrustar otro documento HTML dentro de la página actual. Esto permite mostrar contenido de otras páginas web, aplicaciones, videos o documentos dentro de una página web.

Atributos comunes de Etiqueta iframe

  • src: La URL del contenido que deseas incrustar.
  • width y height: Definen el tamaño del <iframe> .
  • frameborder: Controla si el borde del <iframe> es visible (0 para ocultarlo, 1 para mostrarlo).
  • allowfullscreen: Permite que el contenido se muestre en pantalla completa si es posible.
  • sandbox: Agrega restricciones al contenido incrustado para mayor seguridad (por ejemplo, allow-scripts permite que el <iframe> ejecute JavaScript).
  • loading: Puedes especificar "lazy" para que el <iframe> se cargue solo cuando esté a punto de aparecer en pantalla, lo cual mejora el rendimiento.
