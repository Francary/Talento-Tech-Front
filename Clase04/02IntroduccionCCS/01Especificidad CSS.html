Especificidad en CSS

La especificidad es un conjunto de reglas que el navegador utiliza para decidir qué estilos aplicar cuando existen múltiples reglas CSS que afectan al mismo elemento. Cada selector tiene un nivel de especificidad, y la regla con mayor especificidad se aplicará, independientemente del orden en el que aparezcan las reglas en la hoja de estilos.

Cálculo de la especificidad

  1. Estilos en línea → a:

    • Un estilo directamente aplicado a un elemento en su atributo style="" tiene la máxima prioridad.
    • Puntuación: (1, 0, 0, 0)

  2. Selectores de ID → b:

    • Los selectores basados en el identificador único de un elemento (ID) tienen alta especificidad. (e.g., #miId).
    • Puntuación: (0, 1, 0, 0) por cada ID.

  3. Selectores de clase, pseudoclases y atributos → c:

    • Selectores de clase, pseudoclases (:hover, :nth-child), y selectores de atributo ([type="text"]) tienen especificidad media.
    • Puntuación: (0, 0, 1, 0) por cada clase, pseudoclase o atributo.

  4. Selectores de tipo (etiquetas HTML) y pseudoelementos → d:

    • Los selectores de tipo, es decir, etiquetas HTML (p, div, h1) y pseudoelementos (::before, ::after), tienen la especificidad más baja.
    • Puntuación: (0, 0, 0, 1) por cada selector de tipo o pseudoelemento.
