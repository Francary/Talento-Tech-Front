Unidades de Medidas

En CSS, las unidades de medida son esenciales para definir tamaños, márgenes, fuentes y otros estilos de diseño. Aquí tienes un resumen de las principales unidades que puedes usar:

Unidades Absolutas

Estas unidades son fijas y no dependen de otros elementos o de la pantalla. Son útiles cuando se necesita un tamaño fijo.

  • px: Píxeles, la unidad más común. (Pading: 20px)
  • pt: Puntos (1 pt = 1/72 de pulgada), usada en diseño de impresión. (Pading: 20pt)
  • cm: Centímetros. (Pading: 1cm)
  • mm: Milímetros. (Pading: 15mm)
  • in: Pulgadas (1 in = 2.54 cm). (Pading: 0.5in)

Unidades Relativas

Estas unidades se ajustan en función del tamaño de otros elementos (como el tamaño de fuente del elemento padre) o del viewport.

  • %: Este elemento ocupa el 50% del ancho del contenedor padre.
  • em: El tamaño de fuente es 2 veces el tamaño del elemento padre.
  • rem: El tamaño de fuente es 1.5 veces el tamaño de fuente base.
  • vw: Este elemento ocupa el 50% del ancho del viewport.
  • vh: La altura de este elemento es el 5% del alto del viewport.
  • vmin: El tamaño de este elemento se basa en el 15% del tamaño menor entre el ancho o alto del viewport.
  • vmax: El tamaño de este elemento se basa en el 10% del tamaño mayor entre el ancho o alto del viewport.
