Modelo de Caja y Flexbox

Modelo de Caja

Un "Modelo de Caja" en CSS (Box Model) es un concepto clave en el diseño de páginas web. Describe cómo se representa visualmente cada elemento HTML en el navegador. Cada elemento se considera una caja rectangular que consta de varias áreas o capas que afectan el diseño y el espacio que ocupa en la página.

El modelo de caja CSS está compuesto por las siguientes partes, de adentro hacia afuera:

  1. Contenido (Content):

    • El área donde se muestra el texto, imagen o cualquier contenido del elemento.
    • Puedes ajustar su tamaño usando propiedades como width y height.

  2. Relleno (Padding):

    • El espacio entre el contenido y el borde (border) del elemento.
    • Este espacio es transparente, pero puede expandirse usando la propiedad padding (ej. padding: 20px;).

  3. Borde (Border):

    • Un borde alrededor del contenido y el relleno.
    • Puedes personalizar el grosor, el estilo y el color del borde con la propiedad border (ej. border: 2px solid black;).

  4. Margen (Margin):

    • El espacio exterior alrededor del borde del elemento, separándolo de otros elementos.
    • Puedes ajustar el espacio con la propiedad margin (ej. margin: 10px;).

Flexbox

Flexbox (Flexible Box Layout) es un modelo de diseño en CSS que facilita la disposición y alineación de elementos dentro de un contenedor. Flexbox está diseñado para distribuir el espacio de manera eficiente y alinear los elementos de forma predecible, tanto horizontal como verticalmente, incluso cuando el tamaño de los elementos es dinámico o desconocido.

Conceptos básicos de Flexbox:

    Contenedor Flex: Un elemento se convierte en un contenedor flex al aplicar la propiedad display: flex o display: inline-flex. Todos los elementos directos de este contenedor se convierten en elementos flexibles (flex items).

Propiedades del contenedor Flex:

  1. display: flex: Activa Flexbox para el contenedor.

  2. flex-direction: Define la dirección en la que se colocarán los elementos dentro del contenedor.

    • row (predeterminado): De izquierda a derecha.
    • row-reverse: De derecha a izquierda.
    • column: De arriba a abajo.
    • column-reverse: De abajo hacia arriba.

  3. justify-content: Alinea los elementos a lo largo del eje principal.

    • flex-start (predeterminado): Alinea los elementos al inicio del eje.
    • flex-end: Alinea los elementos al final del eje.
    • center: Centra los elementos a lo largo del eje.
    • space-between: Distribuye los elementos con espacio igual entre ellos.
    • space-around: Distribuye los elementos con espacio alrededor de ellos.

  4. align-items: Alinea los elementos a lo largo del eje transversal.

    • flex-start: Alinea al inicio del eje transversal.
    • flex-end: Alinea al final del eje transversal.
    • center: Alinea en el centro del eje transversal.
    • stretch (predeterminado): Estira los elementos para llenar el contenedor.
    • baseline: Alinea según la línea base del contenido.

  5. flex-wrap: Controla si los elementos se envuelven cuando el espacio en el contenedor es insuficiente.

    • nowrap (predeterminado): Todos los elementos en una sola línea.
    • wrap: Los elementos se envuelven en nuevas líneas si es necesario.
    • wrap-reverse: Lo mismo que wrap, pero con el orden inverso.

  6. align-content: Controla el espacio entre las líneas en un contenedor con varios renglones (cuando flex-wrap está activo).

    • flex-start, flex-end, center, space-between, space-around, stretch.
