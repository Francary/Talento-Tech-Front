Que es Grid?

En CSS Grid Layout (o simplemente CSS Grid) es una herramienta poderosa que permite crear diseños de página en dos dimensiones, organizando el contenido en filas y columnas. CSS Grid ofrece gran control sobre el posicionamiento y alineación de los elementos, lo que facilita crear estructuras complejas de manera ordenada y flexible.

  1. Configuración de un contenedor Grid

    Primero, necesitas un contenedor que actúe como la "grid" principal. Puedes hacerlo de esta forma:

    • Activa el Grid Layout. => display: grid;
    • Define el número y tamaño de las columnas. => grid-template-columns: repeat(3, 1fr); Crea 3 columnas de igual ancho
    • Define el número y tamaño de las filas. => grid-template-rows: auto; Define las filas automáticamente
    • Crea espacio entre las filas y columnas => gap: 10px; (también puedes usar row-gap y column-gap por separado).

  2. Definir el tamaño de las columnas y filas

    En el ejemplo anterior, usamos 1fr para dividir el espacio en fracciones iguales. Aquí hay algunas otras unidades útiles:

    • fr: Fracción del espacio disponible.
    • px o %: Tamaños fijos en píxeles o porcentajes.
    • auto: Ajusta automáticamente el tamaño según el contenido.
