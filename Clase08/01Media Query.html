Media Query

Que es Media Query

Es una técnica que permite aplicar diferentes estilos a una página web según el tamaño de la pantalla o las características del dispositivo. Esto es fundamental para crear sitios web responsivos, que se adapten bien a diferentes tamaños de pantallas, como móviles, tabletas y monitores de computadora.

La sintaxis básica de una media query es la siguiente:

@media (condición) { /* Estilos CSS */ }

Principales condiciones de las media queries

  1. max-width: Aplica los estilos cuando el ancho de la pantalla es menor o igual al valor especificado.
  2. min-width: Aplica los estilos cuando el ancho de la pantalla es mayor o igual al valor especificado.
  3. max-height y min-height: Similar a width, pero basado en la altura de la pantalla.
  4. orientation: Aplica estilos según la orientación de la pantalla (por ejemplo, landscape o portrait).
  5. aspect-ratio: Permite aplicar estilos según la proporción de aspecto de la pantalla.

Ejemplos Media Query

  1. Menor a 600px background-color: darkseagreen
  2. Mayor a 600px y Menor a 1024px background-color:darkslategray
  3. Mayor a 1024px background-color: darkkhaki
