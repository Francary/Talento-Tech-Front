Tipos de Datos

En JavaScript, los tipos de datos son las diferentes categorías en las que se clasifican los valores que se pueden almacenar y manipular en un programa. Los tipos de datos se dividen en dos grandes grupos: primitivos y de referencia.

  1. Tipos de datos primitivos

    Los datos primitivos son inmutables (su valor no puede cambiar directamente) y se almacenan por valor.

      Ver Ejemplos en la consola presiona F12

    1. string (Cadenas de texto) : Representa texto o cadenas de texto.
    2. number (Números) : Incluye enteros y decimales
    3. bigint (Números grandes) : Se usa para representar números enteros muy grandes, más allá del límite de number y Se define agregando una n al final del número.
    4. boolean (Booleanos) : Representa valores de verdad: true o false
    5. undefined : Significa que una variable ha sido declarada pero no tiene un valor asignado
    6. null : Representa una ausencia intencional de valor
    7. symbol :Es un tipo único e inmutable que se usa para crear identificadores únicos

  2. Tipos de datos de referencia

    Los datos de referencia son objetos y se almacenan por referencia (no directamente por valor).

      Ver Ejemplos en la consola presiona F12

    1. Objetos (object):Contienen colecciones de datos o funcionalidades (pares clave-valor).
    2. Arreglos (array) :Una lista ordenada de valores.
    3. Funciones (function) : Bloques de código reutilizables.
